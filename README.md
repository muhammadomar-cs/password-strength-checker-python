# Password Strength Checker (Python)

This project is a simple command-line Python program that evaluates password strength based on common security rules.

## Features
- Checks minimum password length
- Validates presence of:
  - Uppercase letters
  - Lowercase letters
  - Digits
  - Special characters
- Provides immediate feedback via CLI

## Why this project?
Weak passwords remain one of the most common causes of account compromise.  
This project was built to practice Python logic while reinforcing basic cybersecurity concepts related to authentication security.

## Technologies Used
- Python
- Regular Expressions (re module)
